# Day 2 - Loops Programs

This folder contains loop-based Java programs with explanation.

-------------------------------------------------------------------------------------------

1. Factorial
* Description: Calculates factorial of a number using loop.  
* Example:  
5! = 5 × 4 × 3 × 2 × 1 = 120  

-------------------------------------------------------------------------------------------

2. Fibonacci
* Description: Prints Fibonacci series where each number is sum of previous two.  
* Example: 
0 1 1 2 3 5 8 13  

-------------------------------------------------------------------------------------------

3. Reverse Number
* Description: Reverses a given number using while loop.  
* Example:  
Input: 1234  
Output: 4321  

-------------------------------------------------------------------------------------------

4. Palindrome Number
* Description: Checks whether number is same when reversed.
* Example:  
121 → Palindrome  

8-------------------------------------------------------------------------------------------

5. Prime Number
* Description: Checks whether a number is divisible only by 1 and itself.  
* Example:  
7 → Prime  

-------------------------------------------------------------------------------------------

6. Prime Range
* Description: Prints all prime numbers in a given range.  
* Example:  
1–20 → 2 3 5 7 11 13 17 19  

-------------------------------------------------------------------------------------------

7. Sum of Digits
* Description: Adds all digits of a number.
* Example: 
1234 → 1+2+3+4 = 10  

-------------------------------------------------------------------------------------------

8. Count Digits
* Description: Counts total digits in a number.  
* Example:
12345 → 5 digits  

-------------------------------------------------------------------------------------------

9. Multiplication Table
* Description: Prints multiplication table of a number.  
* Example:
5 × 1 = 5  
5 × 2 = 10  

-------------------------------------------------------------------------------------------
10. Armstrong Number
* Description: Checks if sum of cube of digits equals the number.  
* Example: 
153 → Armstrong number
